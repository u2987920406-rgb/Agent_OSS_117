# OSS-117 — Configuration Projet

## Projet
- Nom: Agent OSS-117
- Type: OS Agentique multi-agent
- Version: 0.3.0
- Langage: TypeScript

## Architecture
- Cerveau orchestrateur (LLM via Ollama)
- Routeur avec validation Zod
- Blackboard (stockage JSON persistant)
- Event Bus (communication async)
- Agents specialises: code_writer, terminal_executor, web_scraper, file_reader, grep_search, glob_search
- Ghost QA (audit des resultats)

## Conventions de code
- Typer toutes les fonctions (pas de any)
- Commentaires en francais
- Fichier prefix par module (core/, agents/)
- Validation Zod sur toutes les entrees du cerveau

## Regles de securite
- Ne jamais supprimer de fichiers sans confirmation
- Whitelist pour les commandes terminal
- Anti path-traversal pour code_writer
- Timeout 30s sur les commandes terminal

## Models IA supportes
- Ollama (local): gemma4:12b, gemma4
- OpenAI (cloud): gpt-4o
- DeepSeek (cloud): deepseek-chat
